# Security Policy

## Supported Versions

Currently, only the latest versions of KoBar are being supported with security updates. 

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

Security is a top priority for KoBar. Since the application handles sensitive user data such as clipboard history, API keys (AI Hub), and personal passwords (Snippet Vault), we take all security vulnerabilities very seriously.

**Please DO NOT report security vulnerabilities through public GitHub issues.** If you believe you have found a security vulnerability in KoBar, please report it directly to us by sending an email to:
**contact@ko48.com**

Please include the following information in your email:
* A description of the vulnerability.
* Detailed steps to reproduce the issue.
* The operating system and KoBar version you are using.
* Any relevant screenshots, videos, or code snippets.

We will investigate all legitimate reports and do our best to quickly address any issues. We will reply to your email as soon as possible to acknowledge receipt and outline the next steps. 

Thank you for helping keep KoBar secure!
