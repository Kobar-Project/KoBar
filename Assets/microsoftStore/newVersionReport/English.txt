KoBar v1.0.23: Horizontal Mode, Multi-Monitor Magic & Eye-Tracking 🚀 Latest
This release transforms KoBar into a highly versatile "Swiss Army Knife" that adapts to any desktop setup. The highly anticipated Horizontal Mode and Multi-Monitor support are finally here!

✨ New Features

Horizontal Layout: KoBar now fully supports horizontal orientation. Snap and use it horizontally or vertically based on your workflow.

Multi-Monitor Support: Seamlessly drag the window across different displays without losing focus.

Eye-Tracking Animation: Added a smooth, interactive eye-tracking animation that brings the UI to life (can be easily toggled in settings).

Smart Recentering: Implemented window recentering based on the widget's current position.

🐛 Bug Fixes

Multi-Screen Resolution: Fixed a major bug where the widget would vanish or clip when dragged to secondary monitors.

Edge Detection: Improved real-time edge detection during dragging (resolved vertical edge detection bugs during horizontal drags).

Mini-Mode Positioning: Corrected panel positioning issues when exiting mini-mode.

📝 Infrastructure & Documentation

Funding Configuration: Added official project sponsorship configuration via Open Source Collective (removed old personal Ko-fi and PayPal links).

README Updates: Added macOS download links, contact information, and updated the sponsor section.
